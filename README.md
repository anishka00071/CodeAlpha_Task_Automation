# Task Automation with Python

A Python-based task automation system developed as part of the CodeAlpha Python Programming Internship (Task 3).

## Automated Tasks
- Move all JPG files from one folder to another
- Extract email addresses from a text file
- Scrape the title of a website
- Log all activities with timestamps

## Concepts Used
- File handling
- os / pathlib / shutil
- Regular expressions (re)
- Web requests
- Exception handling

## How to Run
python task_automation.py
